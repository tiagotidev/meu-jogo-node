#  Age of War - Simulador de Evolução

Este é um projeto **Full Stack** simples criado para fins didáticos, simulando a lógica de evolução do clássico jogo "Age of War".

##  Tecnologias
* **Backend:** Node.js com Express
* **Frontend:** HTML5 e JavaScript (Fetch API)
* **Persistência:** File System (fs) para salvar o progresso em JSON.

##  Como rodar o projeto
1. Clone o repositório.
2. No terminal, instale as dependências: `npm install`.
3. Inicie o servidor: `node server.js`.
4. Abra o `index.html` no seu navegador.
